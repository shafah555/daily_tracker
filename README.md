# Daily Tracker

A modern, elegant, and installable daily productivity app designed for Muslim users who want to build a meaningful routine around prayer, Quran, dua, learning, and self-reflection.

This app is more than a simple tracker. It is a polished digital companion that helps users stay consistent with their daily worship and personal growth in a clean, motivating, and mobile-friendly experience.


## How to install as an app

click on          https://shafah555.github.io/daily_tracker/      

On desktop:
- Open the app in Chrome or Edge
- Click the install icon in the address bar

On mobile:
- Open the app in Chrome
- Tap Share
- Select Add to Home screen

## Key features

### 1. Daily report system
Users can record their daily progress in one place, including:
- prayer completion
- Quran pages and minutes
- hadith study count
- Islamic literature time
- general study time
- personal notes

### 2. Dua checklist
A dedicated checklist for daily duas makes it easy to remember and track spiritual habits.

### 3. Quran routine
Users can create a personalized Quran routine and mark progress directly from the daily report page.

### 4. Clean dashboard experience
The app is designed with a calm, premium look that feels welcoming and inspiring for everyday use.

### 5. Progressive Web App (PWA)
The app is built to behave like a real app:
- installable on mobile and desktop
- fast loading
- works offline after first visit
